    "growth_euro" had a value greater than 2 or less than -2
  • Generate a line plot with the "growth_euro" series, label the axes and the title
  • Use the command plt.fill_between() as in Lecture 22 to create a shaded region

# Write your own code
currencies["bool_cond"] = currencies["growth_euro"].between(-2, 2)

plt.plot("date", "growth_euro", data = currencies)
plt.xlabel("Time")
plt.ylabel("Daily percentage change ")
plt.title("Euro Growth Rates Above 2 and Below -2")

vec_y = currencies["growth_euro"]
plt.fill_between(x= currencies["date"],
                 y1 = vec_y.min(),
                 y2 = vec_y.max(),
                 where = -currencies["bool_cond"],
                 alpha = 0.2,color = "red")

plt.show()
